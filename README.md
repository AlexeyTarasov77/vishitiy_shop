## Vishitiy.ua - project

---

### Запуск проекта:

1. ``` git clone https://github.com/danilgubarev/vishitiy_shop.git ```

2. ``` pip install -r requirements.txt ```

3. ``` npm install ```

4. ``` npm run dev ``` (запуск отслеживания изменений для подгрузки классов tailwind)

5. ``` python manage.py runserver ```
